Checkout the features included in Squiz Matrix for creating content.

FUNCTIONALITY

OVERVIEW

User Friendly

Squiz Matrix has two interfaces for accessing and managing content. The first, known as the “Easy Edit Suite”, allows users to edit the content on a single page, within the design of their site. The second is the “Administration Interface”, where site administrators have access to the full range of features.

Design Templates

Squiz Matrix separates content from its presentation through the use of "design" templates, which are used to define the layout and look-and-feel of pages

Content Templates

Squiz Matrix allows you define a Paint Layout, which is similar to a Design for the body of your content, allowing you to customise how individual types of assets should look.

Re-Purposing Content

Squiz Matrix allows site users to define their experience by choosing between multiple designs when visiting your site.

Clean (X)HTML

Web content created in the WYSIWYG editor is filtered through the open source tool HTML Tidy to clean (X)HTML code.

Importing from Microsoft Word

Squiz Matrix allows you to easily cut-and-paste content from Microsoft Office applications (such as MS Word) or HTML generated by other applications (such as Dreamweaver). The structure of tables, bullet text and numbered text is retained, while the styles from your associated stylesheet are applied.

Accessibility - Automatic Validation

Squiz Matrix may be used to publish sites that conform to W3C Web Content Accessibility guidelines. Sites published from the CMS may satisfy all accessibility requirements up to WAI Conformance Level "Triple-A" - Priority 1, 2 and 3 checkpoints. Content may be cleaned by HTML tidy before being stored by the CMS and HTML Tidy includes its own accessibility checks.

Previewing Your Content

Squiz Matrix allows you to preview a page as it will appear in the published site, by selecting Preview for the given page. Previewing the page allows you to see the content with the Design template and styles applied.

Spell Checking

Squiz Matrix provides a visual spell checker as part of the WYSIWYG editor and includes automated suggestions and corrections. The dictionary can be set on the server for different languages, or customised dictionaries.

Search and Replace

Squiz Matrix allows you to search and replace content within a single page, a site sub-section or the entire repository. The Search and Replace tool lists each instance of the located search term, and allows the user select which occurrences to replace. You can also select which asset types and the status of assets to be included in the search.
